
# Forum

Rest API for Forum. Golang without dependencies (almost :D)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Levap123/forum
```

Go to the project directory

```bash
  cd forum
```

Start the server

```bash
  go run cmd/main.go
```
